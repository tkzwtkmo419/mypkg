# スクリプトの説明

* talker.pyからInt16型のメッセージを送信して、listener.pyでInt16型のメッセージを受信、表示するもの。
* 今回は、10秒間ノードを実行して、出力すべきものを出力するプログラム
* 具体例　　grep　"Listen: 10" の場合

```
$../test/test.bash
[listener-2] ...: Listen: 10
```
