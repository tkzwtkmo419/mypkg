# mypkg
* ロボットシステム学の練習リポジトリ
* これは、ROS2のパッケージです。

![test](https://github.com/tkzwtkmo419/mypkg/actions/workflows/test.yml/badge.svg)

# テスト内容の説明

* 10秒間ノードを実行して、出力すべき行を書くというもの
* 実行例　　grep　"Listen: 10" の場合

```

[listener-2] ...: Listen: 10
```

## テスト環境
* Ubuntu 22.04

## 必要なソフトウェア
* Python3.7~3.10
